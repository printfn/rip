# Recursive Interlocking Puzzles

This project uses CMake and GLFW, and requires OpenGL 3.3 to be available.
It is tested on macOS and Linux, and can be compiled with Clang or GCC.

Instructions for building:

```bash
mkdir build
cd build
cmake ..
make
./puzzles
```

Key Bindings:

* Arrow keys to move the camera
* 'F' to advance time forwards
* 'B' to reverse time
* Escape to exit
